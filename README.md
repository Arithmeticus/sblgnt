MorphGNT SBLGNT
===============

Project to merge the MorphGNT analysis with the SBLGNT text.

NOTE: most recent work is taking place on `tisch-merge` branch.

Columns
-------

 * book/chapter/verse
 * part of speech
 * parsing code
 * text (including punctuation)
 * word (with punctuation stripped)
 * normalized word
 * lemma

Parsing Code
------------

 * person
 * tense
 * voice
 * mood
 * case
 * number
 * gender
 * degree
